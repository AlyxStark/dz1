## SimplActNum   
Прогмамм запрашивает 2 числа. После ввода чисел, на экран выводится их:                                                                                                                   
-сумма                                                                                                                                                                                  
-разность                                                                                                                                                                               
-среднее арифметическое                                                                                                                                                                 
-произведение                                                                                                                                                                           
-частное

Программа написана на языке программирования C++, в Visual studio 2019
